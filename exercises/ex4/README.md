# Exercise 4 - Personalize the User Interface for All Users of Your Customer

In this exercise, you personalize the user interface of the **Poetry Slam Manager** for all users of your customer with key user flexibility using SAP Build Work Zone.

> Note: There are some preconditions to enable key user flexibility. You can get more details in chapter [Enable Consumer-Specific Extensions](https://github.com/SAP-samples/partner-reference-application-extension/blob/main/Tutorials/02-DataModelExtensibility.md) of the Partner Reference Application.

## Exercise 4.1 - Adapting the User Interface for All Users

After completing these steps, the **Price** is shown in the **Poetry Slams** table of the **Poetry Slam Manager** for all users:

1. Open the **Poetry Slam Manager** application.
2. Open the **User Action** menu by choosing your profile in the top right corner of the UI.
3. Choose **Adapt UI** to start the **UI Adaptation**.
4. Right-click a colum of the **Poetry Slams** table.
5. Choose **View Settings**.
6. Select the **Price** column.
7. Close the popup.
8. Activate the new version on the top left.
9. Enter a version title, for example **Add Price**.
11. Activate the version with title *AddPrice*.
10. Publish the version.
11. Exit the adaptation mode.
12. In case the **Price** column is not visible, switch the display of the list to show all columns by selecting the **Show More per Row** icon.

## Summary

You've now personalized the **Poetry Slam Manager** UI for all users of your customer.
